## Testing

To test your code locally, run the command:
```
hugo server -D
```


## Delavnice
```
hugo new workshops/YYYY-MM-DD.md
```
Pokaže se na spletni strani, če je datum objave po datumu dogodka --> Spremeni datum objave na po dogosku. Bi moglo delat. Mogoče pa ne bo. Poskus za naslednjič.


Your folder is located at `./your-fork-name/hugo/content/workshops/YYYY-MM-DD.md`.
